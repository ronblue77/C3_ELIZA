# C3_ELIZA
2 versions of ELIZA chatbot implementations in C3

version 1 ./c3_eliza-libc.c3 - a C3 C-like implementation using libc

version 2 ./c3_eliza-std.c3 - a C3 implementation using C3 std (standard library)
